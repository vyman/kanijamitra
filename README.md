# Khanija Mitra - Calcium Granulated Lime Website

A modern, responsive website for promoting premium calcium granulated lime products for agriculture.

## Features

- **Modern Design**: Clean, professional layout with green/earth tone color scheme
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices
- **Smooth Animations**: Hover effects and transitions for better UX
- **SEO Optimized**: Proper meta tags and semantic HTML structure

## Sections

1. **Hero Section**: Eye-catching banner with call-to-action buttons
2. **About Section**: 6 feature cards highlighting product benefits
3. **Products Section**: Detailed product showcase with specifications
4. **Benefits Section**: 6 numbered benefits with descriptions
5. **CTA Section**: Special offer promotion
6. **Contact Section**: Contact form and information
7. **Footer**: Company info and quick links

## Color Scheme

- Primary Green: `#4a7c59`
- Dark Green: `#2d5a3d`
- Accent Green: `#8bc34a`
- Light Background: `#f5f8f5`

## Fonts

- **Headings**: Poppins (Google Fonts)
- **Body**: Roboto (Google Fonts)

## How to Use

1. Open `index.html` in your web browser
2. Customize contact information in the HTML file
3. Replace placeholder product images as needed
4. Update content to match your specific product details

## Customization

### Update Contact Information

Edit the contact section in `index.html`:
- Email: Line 191
- Phone: Line 195
- Address: Line 199

### Change Colors

Modify CSS variables in `style.css` (lines 7-17):
```css
:root {
  --primary-green: #4a7c59;
  --dark-green: #2d5a3d;
  --accent-green: #8bc34a;
  /* ... */
}
```
